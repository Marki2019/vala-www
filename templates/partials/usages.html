What can you build?

GUI Applications
Perfect integration of GTK, Libadwaita and Granite makes development super simple. Publish easily your apps on Flathub without bloated packages.

Command Line Programs
Use a large standard library and use loads of other libraries available for use in Vala. Vala makes also packaging and integration into operating systems easy.
Libraries
Provide bindings from many other languages and API documentation without manual work. Also dynamic and static linking is both supported.
