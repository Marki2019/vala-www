Tooling

Writing

Over the years the community has produced tooling to help make the enhance the experience of writing Vala code like: Vala Language Server, Vala-Lint and Valadoc.

There is support for a wide range of code editors, including: GNOME Builder (which has built-in support for the language), Visual Studio Code, Vim and more.

Build Systems

While there are plenty of great build systems that can be configured to work with Vala, the preferred build system among the community is Meson.
